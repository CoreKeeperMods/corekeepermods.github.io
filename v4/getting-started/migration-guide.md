# Migration Guide from 3.x to 4.x

With many breaking changes in CoreLib 4.x, this page describes steps to migrate mods written for CoreLib 3.x to CoreLib 4.x

# Rename classes
Some classes and functions have been renamed:

- LoadModules function has been renamed to LoadSubmodule
- And CommandsModule has been renamed to CommandModule
- And RewiredExtensionModule has been renamed to ControlMappingModule


This guide is not complete. More info will be added later
