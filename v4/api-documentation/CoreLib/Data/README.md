# `CoreLib.Data` Namespace

> Shared data utilities used by multiple modules.

## Summary

| Type  | Name               | Description                                                                           |
|-------|--------------------|---------------------------------------------------------------------------------------|
| Class | `IdBind`           | ID binding utility used to map strings/types to numeric IDs within configured ranges. |
| Class | `IdBindConfigFile` | Persistent ID binding backed by a config file (used for stable IDs across runs).      |
